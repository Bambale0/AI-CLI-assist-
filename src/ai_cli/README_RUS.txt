# AI CLI Assistant (Русская версия)

**Версия:** 1.4.0

Готовый к промышленному использованию интерфейс командной строки (CLI) для взаимодействия с API, совместимыми с OpenAI и OpenRouter. Этот инструмент позволяет проводить интерактивные беседы с ИИ и работать с файлами в безопасном рабочем каталоге.

---

## Возможности

- **Интерактивный чат**: Непрерывный диалог с моделью ИИ прямо в терминале.
- **Работа с файлами**:
  - `attach` – Добавить содержимое файла к вашему запросу.
  - `create` – Создать новый файл с помощью ИИ по инструкции.
  - `edit` – Изменить существующий файл с помощью инструкций ИИ.
  - `save` – Сохранить последний ответ ИИ в файл.
- **Безопасная рабочая среда**: Все файловые операции ограничены указанным рабочим каталогом для предотвращения случайного выхода за его пределы.
- **Красивый вывод**: Использует библиотеку `rich` для красивого отображения Markdown и форматированных панелей.
- **История сессии**: Сохраняет историю разговора и поддерживает историю команд.
- **Поддержка нескольких провайдеров**: Настроен для работы с любым OpenAI-совместимым API (OpenAI, OpenRouter, локальные хосты).

## Установка

1.  **Убедитесь, что у вас установлен Python 3.8+.**
2.  **Установите пакет с помощью pip:**
    ```bash
    pip install ai-cli-assistant
    ```
    *Примечание: Название пакета выше является условным; используйте актуальное название, если пакет опубликован на PyPI. Alternatively, можно запускать скрипт напрямую, скачав исходный код.*

3.  **Установите необходимые зависимости:**
    Скрипт требует несколько библиотек. Установите их через pip:
    ```bash
    pip install requests python-dotenv rich prompt_toolkit yaspin tqdm
    ```
4. [ pip install . ] В корне проекта

## Настройка

### Переменные окружения

Установите следующие переменные окружения в вашей оболочке или в файле `.env`:

- **API Ключ**: Установите одну из этих переменных вашим API-ключом.
  - `OPENAI_API_KEY`
  - `AI_CLI_API_KEY`
  - `OPENROUTER_API_KEY`

- **Модель и эндпоинт (Опционально)**:
  - `AI_CLI_MODEL`: Используемая модель (по умолчанию: `moonshotai/kimi-k2`).
  - `AI_CLI_API_BASE`: Базовый URL API (по умолчанию: `https://openrouter.ai/api/v1`).

- **Логирование (Опционально)**:
  - `AI_CLI_LOG_LEVEL`: Уровень логирования (`DEBUG`, `INFO`, `WARNING`, `ERROR`). По умолчанию `WARNING`.
